# docker部署renren-fast

### 下载grafana镜像

  docker pull grafana/grafana

### 启动grafana

  docker run --name grafana -p 3000:3000 -d grafana/grafana


### 打开Grafana

地址：http://ip地址:3000/
例如：http://192.168.0.1:3000/

账号：admin，默认密码：admin，点击Log in
